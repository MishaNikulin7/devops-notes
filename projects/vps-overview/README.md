# VPS Infrastructure Overview

## Goal

This project documents my personal VPS environment used for Linux administration and DevOps practice.

## Server

- OS: AlmaLinux 9
- Access: SSH
- Containerization: Docker
- Web server / reverse proxy: Nginx
- Database: MySQL
- Monitoring: Zabbix Server
- Security: Fail2ban
- SSL: TLS certificates configured

## Services

- 3x-ui running in Docker
- Nginx
- MySQL
- Zabbix Server
- Fail2ban
- SSL certificates

## What I practiced

- Linux server administration
- SSH access
- Docker container management
- Nginx configuration
- SSL certificate setup
- MySQL service management
- Zabbix monitoring
- Basic server hardening with Fail2ban
- Git and GitHub workflow

## Notes

Sensitive data such as real IP addresses, passwords, tokens and private keys are not published.
